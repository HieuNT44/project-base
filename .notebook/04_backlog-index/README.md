# Backlog index

Executable tasks live in [`backlogs/`](../backlogs/) at repo root — not inside `.notebook/`.

## Why separate?

| Location | Role |
|----------|------|
| `.notebook/` | Catalogs, design, reports — planning artifacts |
| `backlogs/` | Task specs devs and agents execute daily |
